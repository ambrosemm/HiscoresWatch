# Hiscores Watch

This plugin notifies you in chat when a high-ranking player or a player with 200m XP in a skill enters your area. It's designed to be informative but not intrusive, with highly configurable alerts.

## Features

*   **Rank-Based Alerts:** Get notified when a player with a rank at or below your configured threshold is nearby.
*   **200m XP Alerts:** Receive alerts for players who have achieved the maximum 200,000,000 XP in any skill.
*   **Configurable Alert Color:** Customize the color of the in-game chat alerts to match your preferences or improve visibility.
*   **Player Ignore List:** Add players to an ignore list to prevent receiving notifications for friends, clan mates, or specific individuals.

## Configuration

The plugin offers several options to tailor the alerts to your liking:

*   **Rank Threshold:** (Default: 100) Sets the rank at or below which a player will trigger an alert.
*   **Alert for 200m XP:** (Default: On) Toggles whether to alert for players with 200m XP in a skill.
*   **Ignore List:** A comma-separated list of player names to ignore.
*   **Alert Color:** (Default: Red) Opens a color picker to set the color of the chat message alert.
